**?? Common Scenarios:**

```
Scenario 1: First Time Setup
    ? Run: .\complete-setup.ps1
    ? Wait 2-3 minutes
    ? Done!

Scenario 2: Container Already Exists
    ? Run: docker start sqlserver2022
    ? Wait 10 seconds
    ? Test connection

Scenario 3: Need to Reset Database
    ? Run: .\complete-setup.ps1 -ForceRecreate
    ? Removes old container
    ? Creates fresh database

Scenario 4: Connection Problems
    ? Check: docker ps (is it running?)
    ? Wait: 30-60 seconds after start
    ? Test: .\test-database-connection.ps1
    ? Logs: docker logs sqlserver2022

Scenario 5: Starting Work Each Day
    ? Docker Desktop should auto-start container
    ? If not: docker start sqlserver2022
    ? Open Visual Studio
    ? Start coding!
```
